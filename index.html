Welcome to Kieners secure Webpage

How we completed this task

  • We installed Knot DNS
  • We configured Knot DNS to serve our Zone kiener.sec21.tech
  • We added automatic DNS-Signing to our Knot DNS configuration
  • We configured the Knot DNS server to push the DS record to the sec21.tech zone
  • We installed the NGINX Webserver on our Server
  • We configured our NGINX Webserver to redirect all HTTP requests to HTTPS
  • We generated SSL-Certificates using Let's encrypt and Certbot
